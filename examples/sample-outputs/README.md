# Sample Outputs

These examples show the shape and tone Cubby expects for local generated artifacts. They are not generated from real student data.

Use them as references when building or testing workflows:

* `lesson-pack/main-idea-grade-2/` shows a low-risk lesson pack with validation and handoff artifacts.
* `parent-emails/conference-prep/` shows a review-gated family communication draft with a checklist.

Sensitive or student-specific examples should use fictional, student-neutral labels only.
