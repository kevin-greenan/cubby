# Tools

Tools are small executable helpers that make Cubby more reliable and useful.

Potential tools include workspace validation, manifest inspection, managed-file diff reports, privacy checks, export generation, redaction helpers, artifact indexing and search, template linting, workflow-pack scaffold generation, and handoff generation.

Tools must not make sensitive educational decisions without human review.

Use `pack-design.md` before generating new workflow packs. It exists to keep the pack library high-performing, organized, and tied to real unmet use cases.
